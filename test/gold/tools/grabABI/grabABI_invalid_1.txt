Purpose:
  Fetches the ABI for a smart contract.

Usage:
  grabABI [flags] <address> [address...]

Arguments:
  addrs - list of one or more smart contracts whose ABI to grab from EtherScan (required)

Flags:
  -k, --known         load common 'known' ABIs from cache
  -s, --sol string    file name of .sol file from which to create a new known abi (without .sol)
  -f, --find string   try to search for a function declaration given a four byte code
  -o, --source        show the source of the ABI information (hidden)
  -c, --classes       generate classDefinitions folder and class definitions (hidden)

Global Flags:
  -x, --fmt string     export format, one of [none|json*|txt|csv|api]
  -v, --verbose uint   set verbose level (optional level defaults to 1)
  -h, --help           display this help screen

Notes:
  - Solidity files found in the local folder with the name '<address>.sol' are converted to an ABI prior to processing (and then removed).
